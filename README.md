# Super PvP

WoW addon with quality-of-life features for PvP, including BG call sheet generation, automatic spirit release, honor bar hiding, and countdown timers on queue popups.

## Features

- **Blitz Tactics** - Generates a BG roster call sheet with faction and spec info via `/strat`
- **Auto Release** - Automatically releases spirit when you die in battlegrounds or arenas
- **Hide Honor Bar** - Hides the honor bar in arenas and battlegrounds
- **Queue Timer** - Adds countdown timers to battleground and dungeon queue popups
