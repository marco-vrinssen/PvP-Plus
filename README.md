# PvPlus

Quality-of-life PvP enhancements for World of Warcraft.

## Features

- **Auto Spirit Release**: Automates the release process in BGs.
- **Honor Bar Hiding**: Hides the honor bar when it's not needed.
- **Queue Timers**: Adds countdown timers for PvP queues.
- **BG Roster Copy**: Export the battleground roster for external use.

## Installation

1. Download the latest version.
2. Extract the files to your `Interface/AddOns` folder.
3. Restart World of Warcraft.
