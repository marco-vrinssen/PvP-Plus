# Super PvP

WoW addon with quality-of-life features for PvP, including arena frame cleanup, automatic spirit release, honor bar hiding, and countdown timers on queue popups.

## Features

- **Arena Frames** - Repositions CC remover, debuff frame, and diminish tray on arena frames, hides casting bar
- **Auto Release** - Automatically releases spirit when you die in battlegrounds or arenas
- **Hide Honor Bar** - Hides the honor bar in arenas and battlegrounds
- **Queue Timer** - Adds countdown timers to battleground and dungeon queue popups
