# CleanBars

Minimalist World of Warcraft addon that automatically hides status tracking bars in competitive PvP environments.

## Features

- Automatically hides honor/experience bars in arenas
- Hides status bars in battlegrounds
- Hides status bars in world PvP combat zones
- Automatically restores bars when leaving PvP areas

## Installation

1. Download the addon
2. Extract to `World of Warcraft/_retail_/Interface/AddOns/`
3. Restart WoW or reload UI with `/reload`

## Purpose

Reduces visual clutter during PvP combat by removing the status tracking bar, providing cleaner screen real estate for focus on actual combat.

## Compatibility

- **Interface Version**: 11.2.0 (The War Within)
- **Conflicts**: None known
